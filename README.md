# Data-mining-python
Data mining practicals covering preprocessing, feature selection, dimensionality reduction, imbalanced data handling, association rule mining, and ML pipelines.

The practicals focus on **data aggregation**, **data integration and transformation**, **dimensionality reduction**, **feature engineering**, **handling imbalanced datasets**, **association rule mining**, and **machine learning pipelines with evaluation reports**.

---

## Practical List

### **Practical 1: Data Aggregation & Indexing Operations**
**File:** `Pr-1-Data Aggregation & Indexing Operations.ipynb`  
- Data restructuring using aggregation techniques  
- Grouping data using `groupby`  
- Indexing and slicing for efficient data retrieval  

---

### **Practical 2: Data Integration, Generalization & Transformation**
**File:** `Pr-2-Data Integration, Generalization & Transformation.ipynb`  
- Integrating multiple datasets  
- Data generalization and aggregation  
- Data transformation techniques  
- Scaling using normalization and standardization  

---

### **Practical 3: Dimensionality Reduction using PCA**
**File:** `Pr-3-Dimensionality Reduction using PCA.ipynb`  
- Reducing high-dimensional datasets using PCA  
- Explained variance analysis  
- Simplifying data for efficient modeling  

---

### **Practical 4: Statistical Data Transformations**
**File:** `Pr-4-Statistical Data Transformations.ipynb`  
- Logarithmic transformation  
- Box-Cox transformation  
- Normalization techniques  
- Handling skewed data and stabilizing variance  

---

### **Practical 5: Advanced Feature Selection**
**File:** `Pr-5-Advanced Feature Selection.ipynb`  
- **Filter methods:** Chi-Square, Correlation, Variance Threshold  
- **Wrapper methods:** Forward and Backward Feature Selection  
- **Embedded methods:** Lasso Regularization, Tree-based Feature Importance  
- Feature selection using Logistic Regression and Random Forest  

---

### **Practical 6: Handling Imbalanced Data**
**File:** `Pr-6-Handling Imbalanced Data.ipynb`  
- Understanding class imbalance (majority vs minority class)  
- **Oversampling techniques** to increase minority class samples  
- **Undersampling techniques** to reduce majority class dominance  
- Evaluating classification performance before and after balancing  

---

### **Practical 7: Association Rule Mining**
**File:** `Pr-7-Association Rule Mining.ipynb`  
- Mining frequent itemsets  
- Generating association rules  
- Support, confidence, and lift analysis  
- Market basket analysis using Apriori algorithm  

---

### **Practical 8: ML Pipelines & Model Evaluation Reports**
**File:** `Pr-8-ML Pipelines & Model Evaluation Reports.ipynb`  
- Building end-to-end machine learning pipelines  
- Regression and classification workflows  
- Generating classification and regression evaluation reports  
- Streamlining preprocessing, modeling, and evaluation  

---

## Learning Outcomes
- Perform advanced data preprocessing and transformation  
- Apply dimensionality reduction and feature selection techniques  
- Handle imbalanced datasets using resampling strategies  
- Discover hidden patterns using association rule mining  
- Build reusable ML pipelines with proper evaluation
